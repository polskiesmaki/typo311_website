Support
=======

Find out how you can get support for Yoast SEO!

Slack
-----

Join the `#yoast-seo-for-typo3 <https://typo3.slack.com/archives/C14EQDFL1>`__
channel on the TYPO3 Slack.

If you are not registered yet, `find out more about how to join. <https://typo3.org/community/meet/chat-slack>`_

Github
------

Found a bug in the extension? Please report this in our
`Github repository <https://github.com/Yoast/Yoast-SEO-for-TYPO3/issues>`__.
